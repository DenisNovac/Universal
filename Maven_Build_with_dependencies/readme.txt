Тестовое приложение для оценки полезности maven в сборках проектов.
Полезность велика (мягко говоря).

mvn archetype:generate --- для создания простого каркаса приложения
artifactId --- папка приложения
groupId --- станет пакетом стартового класса, если не указано нижнее
package(grooupId) --- пакет для стартового класса

mvn compile - компилирует все классы в папку target;
mvn package - то же самое плюс jar-файл
mvn clean - очищает папку target
mvn install - то же, что package, но кладёт копию в папку юзера

mvn clean compile assembly:single - если есть плагин встраивания зависимостей, то строит jar-файл, в который упаковывает все зависимые библиотеки (например, дрова для mysql)
